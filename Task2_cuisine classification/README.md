# 🍽️ Task 3: Cuisine Classification

## 📌 Project Overview
This project focuses on classifying restaurant cuisines using Machine Learning techniques. The goal is to predict the cuisine category of a restaurant based on its available features.

## 🎯 Objective
- Build a cuisine classification model.
- Analyze restaurant data.
- Predict cuisine categories using machine learning algorithms.

## 🛠️ Tools & Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## 📂 Dataset
The dataset contains restaurant information such as:
- Restaurant Name
- City
- Locality
- Average Cost for Two
- Rating
- Online Delivery
- Table Booking
- Cuisines
- Votes

## ⚙️ Methodology
1. Loaded and explored the dataset.
2. Handled missing values.
3. Encoded categorical features using Label Encoding.
4. Split the dataset into training and testing sets.
5. Trained a Random Forest Classifier model.
6. Evaluated model performance using Accuracy Score.

## 📊 Results
- Model Used: Random Forest Classifier
- Accuracy: **13.13%**

## ✅ Conclusion
A Random Forest Classifier was developed to classify restaurant cuisines based on restaurant features. The model achieved an accuracy of **13.13%**. This project demonstrates the application of machine learning classification techniques on restaurant datasets.
