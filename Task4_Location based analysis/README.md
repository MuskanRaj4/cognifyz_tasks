# 📍 Task 4: Location-Based Analysis

## 📌 Project Overview
This project analyzes restaurant locations to identify trends in restaurant distribution, ratings, and price ranges across different cities.

## 🎯 Objective
- Identify cities with the highest number of restaurants.
- Analyze average restaurant ratings by city.
- Compare pricing trends across locations.

## 🛠️ Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📊 Analysis Performed
1. Top 10 cities by restaurant count.
2. Average rating analysis by city.
3. Price range comparison.
4. Rating distribution visualization.

## ✅ Conclusion
The analysis provided insights into restaurant density, customer ratings, and pricing patterns across different locations.
